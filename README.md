# Choose Your Win

An interactive game for users to navigate women in the workplace issues and know their rights. 

# Current Features
A user can play the game and navigate the recruitment and hiring process through the character, Rhae.

# Technologies:
* HTML 
* jQuery
* CSS
* flexboxgrid
